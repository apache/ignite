"EXT" FOLDER
------------
This folder can contain JAR libraries to be automatically
loaded when ggstart.{sh|bat} script is used to start up the 
GridGain instance.

Simply put necessary JAR files into ${GRIDGAIN_HOME}/libs/ext and start
the GridGain instance using ggstart.{sh|bat} script and all JAR files 
will be placed on the classpath.
