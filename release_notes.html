Apache IGNITE 2.7.6

Fixed

  • Ignite work directory is now set to the current user's home directory by default, native persistence files will not be stored in the Temp directory anymore [#IGNITE-12057]
  • Fixed a bug that caused a SELECT query with an equality predicate on a part of the primary compound key to return a single row even if the query matched multiple rows [#IGNITE-12068]
  • Fixed an issue that could cause data corruption during checkpointing [#IGNITE-12081]
  • Fixed an issue where a row size was calculated incorrectly for shared cache groups, which caused a tree corruption [#IGNITE-12060]
  • Reduced java heap footprint by optimizing GridDhtPartitionsFullMessage maps in exchange history [#IGNITE-11767]
  • .NET: Native persistence now works with a custom affinity function [#IGNITE-10451]
  • Fixed an issue where an outdated node with a destroyed cache caused the cluster to hang [#IGNITE-9562]
  • Fixed a bug that made it impossible to change the inline_size property of an existing index after it was dropped and recreated with a different value [#IGNITE-12061]
